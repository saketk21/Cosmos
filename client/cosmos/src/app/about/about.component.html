About Cosmos

What is Cosmos?

Cosmos is an expense manager application built using Angular 6, Node.JS and MongoDB. It was developed in an extremely short time span of 2 days. This was built in order to showcase the versatility of Angular in creating engaging UIs.

Developers

Cosmos is a single-person project made by saketk21 (in 2 days), which is a self-brag, but a great feat!

Why Angular?

I chose Angular mainly because I was made to, as this is a demo project I created to showcase my knowledge of Angular 6 before I work on company codebase. However, i am beginning to like Angular, because we can easily re-use Components. As an example of this, the expense-view-component has been re-used on the Manage as well as Analyze pages.
